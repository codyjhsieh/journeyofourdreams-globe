Our Travels, On One Living Globe

Every blog post mapped to its real place on an accurate 3D Earth. Spin the planet, explore a country, and click any pin to open the story.

↑ This is the exact element that embeds into the Squarespace blog. Drag to spin · scroll to zoom · click a pin.

Add it to Squarespace

The globe is a self-contained page hosted on GitHub Pages, dropped into any Squarespace page through a Code Block.

Step 1

Edit a page or blog post → Add Block → choose Code.

Step 2

Paste the snippet on the right. Set the block to HTML (not Markdown).

Step 3

Save. The globe loads inside the page; clicking a pin opens the matching blog post in a new tab.

<iframe
  src="https://codyhsieh.com/journeyofourdreams-globe/embed.html"
  title="Journey of Our Dreams Travel Globe"
  style="width:100%;height:80vh;min-height:560px;border:0;border-radius:16px;overflow:hidden;"
  loading="lazy" allow="accelerometer; gyroscope">
</iframe>

Replace the URL host with your GitHub Pages address after you publish the repo.
